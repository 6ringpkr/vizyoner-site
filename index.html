help Getting Started

Welcome to Morph Messaging!

This app helps you manage real-time notifications from your MQTT broker. Here's how to get started:

1

Configure MQTT Connection

Go to the Config page and set up your MQTT broker details. You'll need the broker URL, topic, and optionally username/password.

2

Enable Push Notifications

Visit the Test page to subscribe to push notifications. This allows you to receive notifications even when the app is closed.

3

Monitor Live Notifications

The Live page shows real-time notifications as they arrive. You can approve, reject, or view details of each notification.

4

Browse History

The History page lets you search and filter through past notifications. Use the search bar and status filter to find specific notifications.

Keyboard Shortcuts

Ctrl+1 Live
Ctrl+2 History
Ctrl+3 Config
Ctrl+4 Test
